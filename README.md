# POS Awesome Lite

![Version](https://img.shields.io/badge/version-23.12.2025-blue)


Point of Sale system for ERPNext v15

## Key Features

-   **Shift Management**: Opening and closing shifts with time controls
-   **Sales Processing**: Create, update, and submit sales invoices
-   **Settlement**: Pay outstanding invoices with payment entry workflow
-   **Customer Management**: Create, search, update customers with credit tracking
-   **Payment Processing**: Multiple payment methods with partial payment support
-   **Draft Invoices**: Save incomplete invoices and continue later
-   **Returns**: Return from invoice or quick return items
-   **Auto-Print**: Automatic printing of closing shift reports
-   **Barcode Scanning**: Fast item selection via barcode
-   **POS Offers**: Apply discounts and offers to invoices

## Technology Stack

-   **Frontend**: Vue.js
-   **Backend**: Frappe/ERPNext Framework
-   **Version**: ERPNext v15
